Instructions

Follow this Quick Guide to start your first website in just few clicks.

Webflow

If you're new to Webflow, take a look at this Webflow 101 Crash Course to understand the basics and essentials.

Customization

To make color, fonts and style modifications easier and quicker, this Template utilizes Webflow variables feature.

SVG

Icons in this template are custom made, SVG code embedded elements. If you want to change their color, select one of them and simply change the font-color on the selection's class.

Buttons

In this template, buttons are components. Each component instance has it's own properties, url and label. These can be edited in the right side menu in Settings, when the instance is selected.
